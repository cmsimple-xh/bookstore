Beschreibung

Erstellen und Verwalten eines Online-Bücher-Katalogs.

Das Plugin kann verwendet werden, um andere Kataloge zu verwalten – Personen, Software usw. Es genügt, die entsprechenden Sprachtexte und die Anzeige in der Konfiguration anzupassen.

Features

  • Buch-Datensätze anlegen
  • Umschlagbild hoch laden
  • Buchsuche nach Schlagwort
  • Rubrik-Filter
  • Konfiguration der Bücher pro Seite
  • Konfiguration der Bild-Uploads
  • Bearbeiten und Löschen von Buch-Datensätzen
  • Hinzufügen und Löschen von Rubriken

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH-Wiki finden Sie weitere Informationen.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Computer.
  3. Laden Sie das gesamte Verzeichnis bookstore/ auf Ihren Server in das CMSimple_XH-Plugin-Verzeichnis hoch.
  4. Vergeben Sie Schreibrechte für die Unterverzeichnisse data/, config/, css/ und languages/.
  5. Gehen Sie zu "Plugins"→"BookStore" im Administrationsbereich, und prüfen Sie, ob alle Voraussetzungen erfüllt sind.

Verwendung

Rufen Sie das Plugin im Template wie folgt auf:

<?php echo bookstore(); ?>

Rufen Sie das Plugin auf einer Seite wie folgt auf:

{{{PLUGIN:bookstore();}}}

Einstellungen

Die Konfiguration des Plugins erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Homepage. Wählen Sie unter "Plugins" "BookStore" aus.

Sie können die Voreinstellungen von BookStore_XH in der Plugin-Konfiguration ändern. Beim Überfahren der Hilfe-Icons mit der Maus werden Hinweise zu den Einstellungen angezeigt.

Die Lokalisierung wird unter "Sprache" vorgenommen. Sie können die Zeichenketten in Ihre eigene Sprache übersetzen, oder sie entsprechend Ihren Anforderungen anpassen.

Das Aussehen von BookStore_XH kann unter "Stylesheet" angepasst werden.

Weitere Einstellungen können in plugins/bookstore/js/bookstore.js vorgenommen werden.

bookstore.js
- Ändern Sie die Textlängenbeschränkungen im /* symbols limit */ Block.
- Stellen Sie Paginierung der Tabelle im /* tablePagination */ Block ein.
  • firstArrow - Bild - Geben Sie ein Bild an, um das Standard-Bild zu ändern. Voreinstellung: (new Image()).src="./path/first.gif"
  • prevArrow - Bild - Geben Sie ein Bild an, um das Standard-Bild zu ändern. Voreinstellung: (new Image()).src="./path/prev.gif"
  • lastArrow - Bild - Geben Sie ein Bild an, um das Standard-Bild zu ändern. Voreinstellung: (new Image()).src="./path/last.gif"
  • nextArrow - Bild - Geben Sie ein Bild an, um das Standard-Bild zu ändern. Voreinstellung: (new Image()).src="./path/next.gif"
  • topNav - Bool'scher Wert - gibt an, ob die Paginierung oben oder unten angezeigt werden soll. Voreinstellung: false
  • rowsPerPage - Zahl - gibt die Anzahl der Zeilen pro Seite an. Voreinstellung: 10
  • currPage - Zahl - gibt die Startseite an. Voreinstellung: 1
  • optionsForRows - Feld - Damit können die Werte für die Zeilen pro Seite Auswahlbox angegeben werden. Voreinstellung: [2.5,10,20,50,100]
  • ignoreRows - Feld - Damit können die zu ignorierenden Tabellenzeilen ('tr') angegeben werden. Es wird empfohlen, dass diese Zeilen unsichtbar sein sollen, da ansonsten die Seitenzählung durcheinander gerät. Voreinstellung: []
- Wenn die Eingabe eines neuen Buchs nicht in einem Dialog erfolgen soll, kommentieren Sie einfach den Code im /* show form as dialog window */ Block aus und entfernen Sie die Kommentare im /* show/hide form only */ Block.

Bücher ohne Umschlagbild
- Wenn das Buch kein Umschlagbild hat, wird statt dessen ein Standard-Bild angezeigt. Selbstverständlich können Sie ein eigenes erzeugen...
- Laden Sie dazu einfach ein Bild mit dem Namen bookstore_nocover.png in das plugins/bookstore/images/ Verzeichnis hoch (nicht plugins/bookstore/data/images/). Wenn das Umschlagbild fehlt, wird das bookstore_nocover.png Bild automatisch ins Zielverzeichnis kopiert.
Anzeigereihenfolge
Beachten Sie, dass die zuletzt hinzugefügten Bücher zuerst angezeigt werden.
Anzeige der Einträge
Die Rubrik, der Buchtitel und Autor werden immer angezeigt. Welche anderen Daten angezeigt werden, ist konfigurierbar.

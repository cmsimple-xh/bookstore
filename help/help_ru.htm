Описание

Плагин предназначен для создания онлайн каталога книг и управления каталогом.

Плагин можно использовать для хранения самых различных данных - картотеки пользователей, программного обеспечения и пр. Для этого достаточно изменить соответствующие языковы строки на нужные и настроить отображение данных в Конфигурации.

Возможности

  • Добавление книг.
  • Загрузка изображения обложки.
  • Поиск книг по ключевому слову
  • Сортировка книг по разделам.
  • Настройка отображения количества книг на странице.
  • Настройка загружаемого изображения.
  • Редактирование и удаление записи.
  • Добавление и удаление разделов.

Установка

Установка плагина не отличается от установки многих других плагинов CMSimple_XH. Более подробно с особенностями установки можно ознакомиться на странице CMSimple_XH wiki.

  1. Создайте резервную копию вашего сайта.
  2. Разархивируйте дистрибутив в папку на вашем компьютере.
  3. Загрузите каталог bookstore/ на серевер в директорию плагинов CMSimple_XH.
  4. Укажите права доступа для подкаталогов data/, config/, css/ and languages/.
  5. Перейдите в режим администрирования "Plugins"→"BookStore" для проверки соответствия требованиям.

Настройки

Конфигурирование плагина в режиме администратора не отличается от конфигурирования большинства других плагинов CMSimple_XH. Выберите "BookStore" в меню "Плагины".

Вы можете изменить настройки BookStore_XH по умолчанию на вкладке "Конфигурация". Поместив курсор мыши на иконке помощи, можно просмотреть подсказки для соотвествующей опции.

Локализация доступна на вкладке "Язык". Вы можете перевести языковые строки на ваш язык, или изменить строки в соответствии с вашими потребностями.

Настройка стиля плагина доступна на вкладке "Стиль".

Другие настройки плагина доступны в plugins/bookstore/js/bookstore.js.

bookstore.js
- Измените ограничение вводимого текста в блоке /* symbols limit */.
- Настройте разбиение записей таблицы в блоке /* tablePagination */.
  • firstArrow - Изображение - используйте свое изображение вместо изображения по умолчанию. По умолчанию: (new Image()).src="./путь/first.gif"
  • prevArrow - Изображение - используйте свое изображение вместо изображения по умолчанию. По умолчанию: (new Image()).src="./путь/prev.gif"
  • lastArrow - Изображение - используйте свое изображение вместо изображения по умолчанию. По умолчанию: (new Image()).src="./путь/last.gif"
  • nextArrow - Изображение - используйте свое изображение вместо изображения по умолчанию. По умолчанию: (new Image()).src="./путь/next.gif"
  • topNav - Логическое - используется для определения расположения блока пагинации - над или под таблицей. По умолчанию: false
  • rowsPerPage - Число - используется для определения начального значения количества записей на странице. По умолчанию: 10
  • currPage - Number - используется для определения начальной страницы. По умолчанию: 1
  • optionsForRows - Массив - используется для задания вариантов количества записей на странице. По умолчанию: [2,5,10,20,50,100]
  • ignoreRows - Массив - используется для определения неотображаемых строк - 'tr' таблицы. Следует использовать для скрытия определенных данных и, соотвественно, разбиения на страницы. По умолчанию: [] (отключено).

Отображение записей
По умолчанию, необходимыми для отображения являются данные о разделе, названии и авторе книги. Отображение других данных настраивается в Конфигурации.

Благодарности

Иконка плагина создана ARTkasimov (akterpnd). Разработчик плагина выражает благодарность за разрешение свободного использования изображения.

Плагин использует свободно распространяемый класс class.upload.php для загрузки и обработки изображений, разработанный Colin VEROT и распространяемый по лицензии GNU GENERAL PUBLIC LICENSE (GPL).

Плагин использует Zebra_Pagination, класс разбиения на страницы, разработанный Stefan Gabos и распространяемый по лицензии LGPL.

Плагин использует библиотеку jQuery для фильтрации ввода в текстовое поле, разработанную Josh Bush и распространяему по особой лицензии.

Плагин использует распространяемую по лицензии MIT библиотеку jQuery Table Pagination для создания постраничной табличной навигации.

Плагин использует иконки, предоставляемые iconsearch.ru.

А также, огромная благодарность Christoph M. Becker, без помощи которого данный плагин мог бы не состояться.

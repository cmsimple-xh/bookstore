Description

Create and manage online book catalogue.

The plugin can be used to store a variety of data - filing users, software, etc. It's enough to set the appropriate language strings on one and customize the display of data in the Сonfig.

Features

  • Create book record
  • Upload cover image
  • Books search by keyword
  • Section filter
  • Book per page setting
  • Uploaded image setting
  • Book record edit and delete
  • Section add and delete

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory bookstore/ to your server into CMSimple_XH's plugin directory.
  4. Set write permissions to the subdirectories data/, config/, css/ and languages/.
  5. Switch to "Plugins"→"BookStore" in the back-end to check if all requirements are fulfilled.

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select "BookStore" from "Plugins".

You can change the default settings of BookStore_XH in "Config". Hints for the options will be displayed when hovering over the help icons with your mouse.

Localization is done in "Language". You can translate the character strings to your own language, or customize them according to your needs.

The style of BookStore_XH can be customized in "Stylesheet".

Further configuration can be done in plugins/bookstore/js/bookstore.js.

bookstore.js
- Change text length limit in /* symbols limit */ block.
- Set table pagination in /* tablePagination */ block.
  • firstArrow - Image - Pass in an image to replace default image. Default: (new Image()).src="./path/first.gif"
  • prevArrow - Image - Pass in an image to replace default image. Default: (new Image()).src="./path/prev.gif"
  • lastArrow - Image - Pass in an image to replace default image. Default: (new Image()).src="./path/last.gif"
  • nextArrow - Image - Pass in an image to replace default image. Default: (new Image()).src="./path/next.gif"
  • topNav - Boolean - used to determine if the pagination should display on top or bottom. Default: false
  • rowsPerPage - Number - used to determine the starting rows per page. Default: 10
  • currPage - Number - This is to determine what the starting current page is. Default: 1
  • optionsForRows - Array - This is to set the values on the rows per page. Default: [2.5,10,20,50,100]
  • ignoreRows - Array - This is to specify which 'tr' rows to ignore. It is recommended that you have those rows be invisible as they will mess with page counts. Default: []
- If you don't wont show new book form as dialog, just comment out code in /* show form as dialog window */ block and uncomment code in /* show/hide form only */ block.

New Book Form
- If you do not want to display add new book form as a dialogue, comment out the code in the /*show form as dialog window */ and uncomment the code in the /* show/hide form only */.
Book without cover
- If book not have cover image, instead default image is displayed. Of course, you may create you own...
- Just upload image with name bookstore_nocover.png to plugins/bookstore/images/ (not plugins/bookstore/data/images/) directory. If cover image is missing, bookstore_nocover.png image copied to destination directory automatically.
Display order
- Note, the last added book is displayed first.
Displaying entries
By default, only data of section, book title and author is nesessary. Displaying of other data is configurable.
